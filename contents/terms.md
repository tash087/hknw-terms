# 箱庭リンク (hknw.link) 利用規約

[英語はこちら](/en/terms)

## 運営者
箱庭スタジオ（HAKONIWA Studio）

---

## 第1条 規約の適用

1. 本規約は、箱庭スタジオ（以下「当運営」といいます）が提供する短縮URLサービス「hknw.link」（以下「本サービス」といいます）の利用条件を定めます。
2. ユーザーは、本サービスを利用することにより、本規約に同意したものとみなされます。
3. 本規約に同意しない場合は、本サービスを利用することはできません。

---

## 第2条 定義

本規約において、以下の用語はそれぞれ次の意味を有するものとします。

| 用語 | 定義 |
|------|------|
| **「短縮URL」** | 本サービスを通じて生成された、転送先URLを指す短いURL文字列（例：https://hknw.link/abc123） |
| **「転送先URL」** | ユーザーが登録したオリジナルのウェブサイトアドレス |
| **「ユーザー」** | 本サービスを利用して短縮URLを生成する個人または法人 |
| **「登録ユーザー」** | 第3条に基づき、メールアドレス認証を行いアカウントを作成したユーザー |
| **「エンドユーザー」** | 短縮URLをクリックして転送先にアクセスする第三者 |
| **「待機画面」** | 短縮URLクリック後に表示される、ガムテープ演出付きの中間ページ |
| **「禁止行為」** | 本規約第4条に定める行為 |

---

## 第3条 ユーザー登録とログイン

1. ユーザーは、当運営指定の方法でメールアドレスを送信し、当該アドレスに送付される「認証コード」を入力することで、ログインおよびユーザー登録を行うことができます。

2. 本サービスは、ログインした「登録ユーザー」に対し、以下の機能を提供します：
   - **データ同期**: 同一メールアドレスでログインした複数デバイス間での、作成済み短縮URLリストの共有
   - **統計管理**: 各短縮URLの統計情報の継続的な確認

3. ユーザーは、自身のメールアドレスおよび送付された認証コードを厳重に管理するものとします。第三者によるメールアドレスの不正利用等により生じた損害について、当運営は一切の責任を負いません。

---

## 第4条 禁止行為

ユーザーは、以下の目的または方法で本サービスを利用してはなりません。

### 4.1 違法コンテンツへの転送

- 児童ポルノ、わいせつ物の販売・頒布に関するサイト
- 著作権、商標権、特許権など第三者の知的財産権を侵害するコンテンツ
- ハッキング、クラッキング、マルウェア配布に関するサイト
- 違法なギャンブル、無許可のオンラインカジノ

### 4.2 迷惑行為・スパム

- スパムメール、チェーンメール、迷惑メールでの短縮URLの使用
- 不正なクリック数の水増し行為（ボットや自動スクリプトによる連続アクセス）
- サービスを妨害するような大量リクエスト（DDoS攻撃など）
- 第三者に迷惑をかける形での短縮URLの拡散

### 4.3 詐欺・なりすまし

- フィッシングサイトへの転送（ログイン情報を盗むサイト等）
- 正当な企業や団体になりすましたURLの作成
- 金融詐欺、投資詐欺、ロマンス詐欺などの犯罪行為
- 偽の懸賞やプレゼントを装った情報収集

### 4.4 公序良俗に反する行為

- 差別的表現やヘイトスピーチ
- 暴力、殺人、自殺を助長するコンテンツ
- 違法薬物等の取引情報
- 虐待行為の描写

## 第5条 利用停止・URL無効化

当運営は、ユーザーが以下のいずれかに該当すると判断した場合、事前通知なく短縮URLの無効化、利用停止、または関連する統計データの削除を行うことができます。

1. 第4条（禁止行為）に違反した場合
2. 第三者から正当な苦情や権利侵害の申告があった場合
3. 当運営からの連絡に正当な理由なく応じない場合
4. その他、本サービスの運営に支障をきたす行為があった場合

無効化された短縮URLへのアクセスは、エンドユーザーに対してエラーページ（404または無効通知）を表示します。

## 第6条 統計データの取り扱い

1. 本サービスは、短縮URLへのアクセス情報として以下を収集します：
   - アクセス日時
   - アクセス元の国・地域（Cloudflareの提供データに基づく）
   - リファラー（アクセス元サイト）
   - クリック数（総数）

2. **登録ユーザー**は、自身の管理画面から、同一メールアドレスで作成した短縮URLの統計情報を閲覧・管理できます。

3. 個人を特定できる情報（IPアドレス、ブラウザフィンガープリント、デバイスID、Cookieなど）は、本条の統計目的では**収集しません**。詳細は[プライバシーポリシー](/privacy)をご参照ください。

## 第8条 広告表示について

1. **フリープラン**では、短縮URLアクセス時に「待機画面」内で広告が表示される場合があります。
2. 広告の内容は、当運営または提携広告主が提供するものです。当運営は広告内容について一切の責任を負いません。
3. ユーザーは、広告表示を回避する手段（広告ブロッカー等）を使用しないことに同意するものとします。
4. **スタジオプラン**では、広告非表示オプションが提供されます。
