## 第1条 規約の適用

1. 本規約は、箱庭スタジオ（以下「当運営」といいます）が提供する短縮URLサービス「hknw.link」（以下「本サービス」といいます）の利用条件を定めます。
2. ユーザーは、本サービスを利用することにより、本規約に同意したものとみなされます。
3. 本規約に同意しない場合は、本サービスを利用することはできません。

---

## 第2条 定義

本規約において、以下の用語はそれぞれ次の意味を有するものとします。

| 用語 | 定義 |
|------|------|
| **「短縮URL」** | 本サービスを通じて生成された、転送先URLを指す短いURL文字列（例：https://hknw.link/abc123） |
| **「転送先URL」** | ユーザーが登録したオリジナルのウェブサイトアドレス |
| **「ユーザー」** | 本サービスを利用して短縮URLを生成する個人または法人 |
| **「エンドユーザー」** | 短縮URLをクリックして転送先にアクセスする第三者 |
| **「待機画面」** | 短縮URLクリック後に表示される、ガムテープ演出付きの中間ページ |
| **「禁止行為」** | 本規約第3条に定める行為 |

## 第3条 禁止行為

ユーザーは、以下の目的または方法で本サービスを利用してはなりません。

### 3.1 違法コンテンツへの転送

- 児童ポルノ、わいせつ物の販売・頒布に関するサイト
- 著作権、商標権、特許権など第三者の知的財産権を侵害するコンテンツ
- ハッキング、クラッキング、マルウェア配布に関するサイト
- 違法なギャンブル、無許可のオンラインカジノ

### 3.2 迷惑行為・スパム

- スパムメール、チェーンメール、迷惑メールでの短縮URLの使用
- 不正なクリック数の水増し行為（ボットや自動スクリプトによる連続アクセス）
- サービスを妨害するような大量リクエスト（DDoS攻撃など）
- 第三者に迷惑をかける形での短縮URLの拡散

### 3.3 詐欺・なりすまし

- フィッシングサイトへの転送（銀行、クレジットカード、SNS等のログイン情報を盗むサイト）
- 正当な企業や団体になりすましたURLの作成
- 金融詐欺、投資詐欺、ロマンス詐欺などの犯罪行為
- 偽の懸賞やプレゼントを装った情報収集

### 3.4 公序良俗に反する行為

- 人種、民族、宗教、性的指向などに基づく差別的表現やヘイトスピーチ
- 暴力、殺人、自殺を助長または賛美するコンテンツ
- 違法薬物、危険ドラッグ、向精神薬の取引情報
- 虐待行為（児童虐待、動物虐待など）の描写

## 第4条 利用停止・URL無効化

当運営は、ユーザーが以下のいずれかに該当すると判断した場合、事前通知なく短縮URLの無効化、利用停止、または関連する統計データの削除を行うことができます。

1. 第3条（禁止行為）に違反した場合
2. 第三者から正当な苦情や権利侵害の申告（DMCA通報を含む）があった場合
3. 当運営からの連絡に正当な理由なく応じない場合
4. その他、本サービスの運営に支障をきたす行為があった場合

無効化された短縮URLへのアクセスは、エンドユーザーに対してエラーページ（404または無効通知）を表示します。

## 第5条 統計データの取り扱い

1. 本サービスは、短縮URLへのアクセス情報として以下を収集します：
   - アクセス日時
   - アクセス元の国・地域（Cloudflareの提供データに基づく）
   - リファラー（アクセス元サイト）
   - クリック数（総数）

2. 個人を特定できる情報（IPアドレス、ブラウザフィンガープリント、デバイスID、Cookieなど）は**収集しません**。詳細は[プライバシーポリシー](/jp/privacy)をご参照ください。

3. 収集された統計データは、以下の目的で使用されます：
   - サービスの改善・最適化
   - 不正アクセスの検出
   - 広告効果測定（広告主契約時のみ、集計値として）

3. ユーザーは、広告表示を回避する手段（広告ブロッカー等）を使用しないことに同意するものとします。
4. 有料プラン（別途契約）では、広告非表示オプションを提供する場合があります。
